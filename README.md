# Repository REYES-CALDAS, MONTENEGRO, et al., 2026	
This repository contains the data and scripts used in the analyses presented in the manuscript. The contents are described below:

- ps_ITS_clean.Rds: RDS file containing the phyloseq object generated from ITS amplicon sequencing data.
- ps_clean.Rds: RDS file containing the phyloseq object generated from 16S rRNA gene amplicon sequencing data.
- orquideas_analysis.Rmd: R Markdown file including all analyses performed in R, along with the code used to generate the figures.
- Supplemental_material: Directory containing all supplementary materials associated with the manuscript.
